Object класс находится в пакете java.lang. Каждый класс наследуется от класса Object неявно. Если класс не наследуется от какого либо другого класса то значит он наследуется от класса Object. Исходя из этого понятно, что все методы класса Object будут наследоваться в вашем классе.

Методы класса Object:
  1. getClass():
  2. hashCode():
  3. equals():
  4. clone():
  5. toString():
  6. notify():
  7. notifyAll():
  8. wait(long timeout):
  9. wait(long timeout, int nanos):
  10. wait():
  11. finalize():
